
# 모델
- config : 경로, 모델명, 모델 파라미터 관련 내용
- models : timm 모델 load
- model_CLIP :  hugging_face 모델 load
- data_loader : 데이터셋 구축
- model_selector : 모델 선택
- transform : 이미지 증강
- utils : 정확도
- trainer : 모델 학습 및 검증
- main : 종합
